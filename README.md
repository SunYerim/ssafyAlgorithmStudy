###  ⭐️ 소개

- 2024.01.25 ~
- SSAFY 부울경 4반 알고리즘 스터디 입니다.
- 자료구조와 알고리즘 개념 공부 및 문제 풀이 후
- 모의 코딩테스트 환경 조성하여 스터디 진행 예정 

### ✔️ 진행방식

- 매주 목요일 8시 디스코드 점검
- 일주일 진행 상황 체크 후
- 다음주 알고리즘 개념 발표 
- 문제 푼 것 토의
### ✔️ 규칙

- 매주 5문제 이상 commit (공통1, 고정2, 자율2)
-   벌점 제도 도입 -> 15점 넘기면 퇴출
  - 지각 -1, `무단불참 -6`, `발표 준비 x -3`, 안 풀어올 경우 -2
  - 매달 벌점 최고점자는 커피쏘기
