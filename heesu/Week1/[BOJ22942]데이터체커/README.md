### 접근방법
겹치는 원이 없도록 -> X축 좌표로 보면 중심이 x, 반지름이 r인 원의 경우 x축 좌표는 x-r, x+r <br>
배열에 각 x 좌표들을 넣고, arr[idx] < x-r < x+r < arr[idx+1] 인 경우를 확인해서 Yes or No 판별하도록 함

-> 문제점: 같은 원인지, 서로 다른 원인지 확인 안 함. 한참을 삽질함

같은 원인지에 대한 정보도 함께 넣어서 관리해야 되겠다. 다시 시도할 예정ㅜ 부들부들
